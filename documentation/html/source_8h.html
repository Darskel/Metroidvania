Charge l'état de la partie.

Les paramètres numSauv et nomObjs doivent être définis avant, les autres paramètres seront remplis par cette fonction

Paramètres
numSauvnuméro de la sauvegarde (entre 1 et 3)
sallenom de la salle à charger
persopersonnage (structure personnage_t) à remplir
inventairetableau contenant l'état de l'inventaire du personnage
nomObjstableau contenant le nom des objets de l'inventaire dans l'ordre
Renvoie
un code erreur en fonction de l'erreur: -1 = Mauvais numéro de sauvegarde, -2 = impossible d'ouvrir ou créer le fichier; le numéro de sauvegarde dans laquelle la sauvegarde a été effectuée (avec succès), ou 0 si le fichier de sauvegarde est vide
